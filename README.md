# 🚢 Трекер поставок Китай → Россия

Веб-приложение для учёта поставок: загрузка накладных, расчёт курсов, отслеживание доставки, подсчёт прибыли.

---

## 🚀 Деплой на Vercel (15 минут)

### Шаг 1 — Загрузить код на GitHub

1. Зайдите на [github.com](https://github.com) и создайте аккаунт (если нет)
2. Нажмите **New repository**, назовите `china-tracker`, нажмите **Create**
3. Загрузите все файлы этой папки в репозиторий (кнопка **uploading an existing file**)

### Шаг 2 — Создать аккаунт Vercel

1. Зайдите на [vercel.com](https://vercel.com)
2. Нажмите **Sign Up** → **Continue with GitHub**
3. Подтвердите доступ к GitHub

### Шаг 3 — Задеплоить проект

1. В Vercel нажмите **Add New → Project**
2. Найдите репозиторий `china-tracker`, нажмите **Import**
3. Нажмите **Deploy** (настройки трогать не нужно)
4. Подождите ~2 минуты — проект задеплоится

### Шаг 4 — Подключить базу данных (Vercel KV)

1. В Vercel откройте ваш проект → вкладка **Storage**
2. Нажмите **Create Database → KV**
3. Назовите `shipments-db`, выберите регион ближайший (Frankfurt)
4. Нажмите **Create & Connect** — переменные добавятся автоматически

### Шаг 5 — Добавить API-ключ Anthropic

1. Зайдите на [console.anthropic.com](https://console.anthropic.com)
2. **API Keys → Create Key**, скопируйте ключ (начинается с `sk-ant-...`)
3. В Vercel: **Settings → Environment Variables**
4. Добавьте: Name = `ANTHROPIC_API_KEY`, Value = ваш ключ
5. Нажмите **Save**

### Шаг 6 — Передеплоить

1. В Vercel: **Deployments → три точки у последнего → Redeploy**
2. Через минуту приложение готово!

## 🔗 Результат

Вы получите ссылку вида `china-tracker.vercel.app` — её можно открыть с любого устройства, данные общие для всех, кто имеет ссылку.

## 📱 Как пользоваться

| Статус | Что делать |
|--------|-----------|
| **Новый** | Создать поставку, загрузить накладную |
| **В пути** | Нажать «Груз прибыл» когда придёт |
| **Прибыл** | Ввести доп. расходы и стоимость доставки |
| **Продан** | Ввести продажную цену → увидеть прибыль |

## ❓ Частые вопросы

**Q: Приложение видно всем?**  
A: Только тем, у кого есть ссылка. Для защиты паролем напишите разработчику.

**Q: Накладная не распознаётся**  
A: Попробуйте сделать фото в хорошем освещении или сохранить как PDF. Можно всегда ввести данные вручную.

**Q: Сколько стоит?**  
A: Vercel и KV бесплатны в рамках лимитов (достаточно для сотен поставок). Anthropic API — копейки за запрос.
